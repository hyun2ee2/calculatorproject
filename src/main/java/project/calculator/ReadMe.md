# 내일배움캠프 계산기 과제 ReadME

---

안녕하세요 내일배움부트캠프 Spring3기 수강생 홍성현 입니다.
이번 과제는 이전 과제의 2차 과제로 부족한 점을 다듬고자 다시 재작성한 계산기 과제 입니다.
총 3개의 STEP으로 구성된 계산기 프로그램 구현 과제로, STEP1에서는 사칙연산 기능을 갖춘 계산기를구현하고,
STEP2에서는 STEP1에서 구현한 계산기를 객체지향적으로 개선하여 클래스를 분리하고 각 클래스의 역할을 명확히 나누는 것을 목표로 작성.
STEP3에서는 더 나아가 객체지향 프로그래밍의 핵심인 제네릭, 람다, 스트림, Enum을 활용하여 계산기를 리팩토링 하였습니다.

---

## 주요 기능

- 처음 사용자로 부터 두개의 값과 연산자를 입력 받은 후 계산을 진행
- 계산 결과를 저장하고 이전 계산 결과를 조회할 수 있음
- 소수점 계산을 지원합니다.

---

## 프로젝트를 통해 배운점

- 제네릭, Enum, 람다, 스트림, 이 네가지의 각 개념의 역할과 사용 목적을 알 수 있었습니다.
